# DFGraph

Is a simple graph based database library based on the `sqlalchemy` library.


## Installation

`pip install dfgraph`

## Testing
till `pip 21.3`:
```
pip install --use-feature=in-tree-build -e .
pytest
```
`pip 21.3+`:
```
pip install .
pytest
```

## Usage

`dfgraph` comes with two main classes: `Node` and `Relationship`.
